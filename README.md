Jump Game
A simple game where you have to jump over the enemy to avoid colliding with it.

How to Play
Open the index.html file in your browser.
Click on the screen to jump over the enemy.
Try to avoid colliding with the enemy.
Notes
The game stops when you collide with the enemy.
You can restart the game after it's over.
Source Code
The source code is written in JavaScript, HTML, and CSS.
